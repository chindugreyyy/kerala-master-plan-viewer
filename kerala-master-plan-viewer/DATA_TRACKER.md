## How to Update This Tracker

When you acquire a new plan:

1. Update the status column (🔴 → 🟡 → 🟢 → ✅)
2. Fill in the document type (e.g., "DP 2021", "PDF", "Scan")
3. Note the source (e.g., "Trivandrum Corp Office", "RTI response")
4. Add contact information
5. Record date sourced
6. Note georeferencing method used
7. Add URL once uploaded
8. Add any notes

| RMP | Regional Master Plan | Wider area plan |
